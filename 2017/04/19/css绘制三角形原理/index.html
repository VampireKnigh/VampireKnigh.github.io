css绘制三角形原理及各种几何图形实现

在写网页的过程,必不可免的会碰到网站需要装饰一些几何图形;
实现的方式无非四种:

  1. 图片
  2. CSS2.1&3
  3. SVG
  4. Canvas

就我个人而言,对于一些常见的几何图形.

  • 图片是不推荐的[太占资源,也不利于维护]
  • CSS是比较推荐的,精简且方便维护
  • SVG也是一个很好的选择[学习成本高一点点]
  • Canvas看情况使用
